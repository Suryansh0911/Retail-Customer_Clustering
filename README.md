# Retail Store Customer Clustering System

### Project Overview
Applied unsupervised learning to segment customers based on purchase patterns from an Online Retail dataset.
- **Data Size:** 500,000+ rows (cleaned to remove 135,080 anonymous IDs).
- **Methodology:** RFM Analysis, Log Transformation, StandardScaler, PCA (2D), and K-Means Clustering ($k=3$).

### Tech Stack
- **Language:** Python 3.11
- **Libraries:** Pandas, Scikit-Learn, Seaborn, Matplotlib
- **Deployment:** Streamlit

### How to Run
1. Clone the repo: `git clone <your-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
